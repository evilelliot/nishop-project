# Getting the project structure
```bash
git clone https://github.com/evilelliot/nishop-project.git
cd nishop-project/nishop
```

# Running systems

For the API:
```bash
cd api
source ./env/bin/activate
pip install -r requirements.txt
python app.py
```

For the client:
```bash
cd client
source ./env/bin/activate
pip install -r requirements.txt
python app.py
```
